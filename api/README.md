# VersoView Backend

## Setup

- Node: v22
- npm: 9.6.1

Add the values to a `.env` file

```env
NODE_ENV = production
PORT = 5001
MONGO_URI_2 =

# AWS IAM Config
AWS_ACCESS_KEY_ID =
AWS_SECRET_KEY =

# AWS S3 Config
AWS_S3_BUCKET_NAME =
AWS_S3_BUCKET_PROFILE =
```

Install & Start server

```
npm i

npm start
```
